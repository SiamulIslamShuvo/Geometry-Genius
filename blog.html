Question: What is DOM and what is the purpose of DOM

DOM stands for Document Object Model.The purpose of the DOM is to dynamically access and manipulate the content, structure, and style of a document. The DOM is essential for modern web development, as it enables us to create dynamic and interactive web pages and web applications. It provides a way for browsers to represent and manipulate the content of a web page, which allows for cross-browser compatibility and makes it easier for us to write code that works consistently across different browsers and platforms.

Question: How will you select HTML elements using DOM? Name the DOM methods.

To select HTML elements using DOM, We can use a variety of DOM methods.

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()

Question: What is event bubble?

When an event is triggered on an element, it will first be processed by the element itself, and then it will be passed to its parent element.The parent element will do the same and pass the event to its own parent. This means that if you have an event listener attached to a parent element, it will be triggered when an event bubbles up from one of its child elements.

Question: What is a callback function and why will you use it?

In JavaScript, a callback function is a function that is passed as an argument to another function, and it is intended to be called or executed by the other function at a later time or in response to some event. We can use callback functions for Event handling,Asynchronous programming,Iteration,Error handling,Generalizing a function etc.
